About This Project

  • Source: Indiancine.ma: an open archive of Indian cinema. Data fetched via their public JSON API.
  • Filtering: Films were queried directly by language = "Telugu" using the indiancine.ma API, which tags each film with its original production language.
  • Scope: Telugu films from 1930–2024, capped at 100 per decade using a spread sample so every year in the decade is represented.
  • Posters: Fetched directly from indiancine.ma's image CDN and cached locally.
  • Synopses: Real plot summaries from the indiancine.ma database where available.
